デイトレード DAY TRADING → 詳細は daytrade-glossary.html
板読みいたよみ
Order Book Reading
基礎
売り板・買い板の数量・価格分布を見て需給バランスや価格の方向性を判断する技術。大口注文（壁）の存在・消え方がエントリータイミングの手がかりになる。
ボラティリティ
Volatility / 価格変動率
価格
株価の価格変動の大きさ・激しさを表す指標。高いほど短期間で大きく動く。デイトレーダーにとって高ボラは利益機会だが、リスク管理も重要。VIX指数は米国市場全体のボラティリティを示す。
出来高できだか
Trading Volume
価格
一定期間に売買が成立した株数。出来高の急増は「相場の転換点」や「トレンドの強さ」のシグナル。価格変動と出来高はセットで分析する。出来高なき上昇は続かないことが多い。
移動平均線いどうへいきんせん
Moving Average / MA
テクニカル
過去N日間の終値の平均をつないだ線。トレンドの方向性と支持・抵抗の把握に使う。5日・25日・75日・200日が代表的。SMA（単純平均）とEMA（指数平滑移動平均）がある。
ゴールデンクロス・デッドクロス
Golden Cross / Dead Cross
テクニカル
GC（ゴールデンクロス）：短期MAが長期MAを上抜ける。上昇トレンド転換のシグナル。DC（デッドクロス）：短期MAが長期MAを下抜ける。下落トレンド転換のシグナル。ダマシも多いため他指標と組み合わせる。
RSI
Relative Strength Index / 相対力指数そうたいりょくしすう
テクニカル
0〜100で表される買われすぎ・売られすぎを測る指標。70超で買われすぎ（過熱）、30未満で売られすぎ（売られ過ぎ）の目安。14日RSIが標準。逆張りエントリーの判断に使われる。
MACD
Moving Average Convergence Divergence
テクニカル
2本のEMAの差（MACDライン）とそのシグナルラインを使うトレンド系指標。MACDがシグナルを上抜け→買いシグナル、下抜け→売りシグナル。ゴールデンクロスと組み合わせると精度が上がる。
ボリンジャーバンド
Bollinger Bands
テクニカル
移動平均線の上下に標準偏差×2の「バンド」を描いた指標。価格が±2σ内に収まる確率は約95%。バンドの収縮後の拡大がトレンド発生のシグナル。バンドウォーク（片側を沿う動き）は強いトレンドを示す。
VWAP
Volume Weighted Average Price / 出来高加重平均価格できだかかじゅうへいきんかかく
テクニカル
その日の取引を出来高で加重した平均価格。機関投資家の取引コスト基準として使われる。デイトレでは「VWAPより上なら強い・下なら弱い」の判断軸として活用。
サポート・レジスタンス
Support / Resistance
テクニカル
サポート（支持線）：価格が下がるときに止まりやすい価格帯。レジスタンス（抵抗線）：価格が上がるときに止まりやすい価格帯。ブレイクアウト（突破）するとトレンドが加速しやすい。
損切りそんぎり
Stop Loss / Cut Loss
資金管理
想定に反した値動きになったとき、損失を確定させてポジションを手放す行為。「損切りは最小限の損失で済ませるための保険」。損切りできない→含み損拡大→退場、という最悪のパターンを防ぐために必須。
基本：エントリー価格の-1〜-3%以内を損切りラインに設定
利確りかく（利食い）
Take Profit
資金管理
含み益が出ているポジションを決済して利益を確定させること。「利食い千人力」——含み益は確定するまで利益ではない。利確ルールを事前に決め、感情で引き延ばさないことが大切。
リスクリワード比
Risk-Reward Ratio / R:R
資金管理
「取るリスク（損切り幅）に対して期待できるリターン（利益目標）の比率」。最低でも1:2以上を目標にすること。1:1以下のトレードは勝率50%でも利益が出ない。
例：損切り-10円・利確+20円 = リスクリワード比 1:2
ナンピン
Averaging Down
注文
含み損が拡大する中で追加買いをして平均取得価格を下げる行為。上手くいけば損失を圧縮できるが、失敗すると損失が倍増する「最大のリスク行動」。デイトレでは原則禁止とする上級者が多い。
信用取引しんようとりひき
Margin Trading
制度
証券会社に担保を預けて、資金の最大3.3倍まで株を売買できる仕組み。空売り（株を借りて先に売る）も可能。レバレッジが利益も損失も増幅する。追証（追加担保）が発生すると強制決済になる。
FOMO
Fear Of Missing Out / 乗り遅れ恐怖
心理
急騰する株を見て「乗り遅れてしまう」という焦りで高値掴みしてしまう心理バイアス。FOMOでのエントリーは損切りになりやすい。「次の機会もある」という心理的余裕が重要。
ダウ理論
Dow Theory
テクニカル
チャールズ・ダウが提唱した相場分析の基礎理論。「高値と安値が切り上がっている間は上昇トレンド、切り下がっている間は下降トレンド」。高値更新の失敗→トレンド転換の警戒シグナル。
ポジションサイズ
Position Sizing
資金管理
1回のトレードに投入する資金量の決め方。1回の損失が総資金の1〜2%以内になるよう逆算してロット数を決める。大きすぎるポジションは感情的な判断を引き起こす。
総資金100万円・損切り幅3%の場合 → 最大損失2万円なら投入額は67万円以下
最大ドローダウン（MDD）
Maximum Drawdown
KPI
資産のピークから底までの最大下落率。自分のシステムのMDDを把握しておくことで、連敗中でも「想定内か想定外か」を判断できる。MDD-30%超なら資金管理を見直す目安。
ギャップ（窓）
Gap / Price Gap
価格
前日の終値と当日の始値の間に生じる価格の空白。上方向のギャップ（窓開け上昇）は強気シグナル。下方向は弱気シグナル。「窓埋め」（ギャップを埋める値動き）が起きることも多い。
セクター（業種）
Sector / Industry
市場
株式市場を事業内容で分類したグループ。日本では電機・自動車・半導体・金融・防衛・化学など。マクロ環境に応じて強いセクターが変わる（セクターローテーション）。その日の「テーマ」を読むのがデイトレの肝。

新NISA
New NISA / 少額投資非課税制度
制度
2024年開始の非課税投資制度。投資利益・配当に税金がかからない。年間360万円・生涯1,800万円・非課税期間無期限・売却後の枠復活が特徴。両枠（積み立て＋成長）を同時利用可能。
積つみ立たて投資枠
Tsumitate Frame / 年120万円
制度
年間120万円・定期定額の積み立てのみ対応。金融庁認定の低コスト・長期分散型商品のみが対象。eMAXIS Slim全世界株式・S&P500などが代表的。手数料が低く長期複利に最適。
成長投資枠
Growth Frame / 年240万円
制度
年間240万円・積み立て＆スポット購入どちらも対応。個別株・ETF・REIT・投資信託が対象。生涯上限は1,200万円。高配当株の長期保有で配当を非課税受取りする戦略に最適。
インデックスファンド
Index Fund
商品
株価指数（S&P500・MSCI ACWIなど）に連動することを目指す投資信託。手数料が低く長期的にアクティブを上回ることが多い。NISA積み立ての最適解として広く支持されている。
複利ふくり
Compound Interest
運用
利益を再投資して「利益が利益を生む」効果。72÷年利率＝資産が2倍になる年数（72の法則）。年利7%なら約10年で2倍。NISAで非課税のまま複利運用することで最大化できる。
ドルコスト平均法
Dollar-Cost Averaging / DCA
戦略
毎月一定額を定期購入する方法。高いときは少量・安いときは多量を自動的に購入するため平均取得単価が平準化される。積み立てNISAが自動的にこの手法を実践する。
リバランス
Rebalancing
管理
資産配分が目標からずれたとき元の比率に戻す作業。年1〜2回が目安。NISAでは売却せず積み立て先を調整する「入金リバランス」が枠を消費しない最適な方法。
4%ルール / 出口戦略
4% Rule / Exit Strategy
出口
資産の4%を毎年引き出しても30年以上枯渇しないという研究に基づく出口目安。2,000万円→年80万円（月6.7万円）を非課税で引き出せる。暴落時に売らないためのキャッシュクッション準備が前提。
信託報酬しんたくほうしゅう
Management Fee / Expense Ratio
コスト
保有中に毎日自動的に引かれる運用管理コスト（年率）。0.1%の差が30年で数十万円の差になる。NISA積み立て枠の目安は年0.2%以下。eMAXIS Slimシリーズは0.05〜0.1%台が多い。
MSCI ACWI（オルカン）
MSCI All Country World Index
指数
世界47カ国・約3,000銘柄をカバーする株価指数。eMAXIS Slim全世界株式（オール・カントリー）が連動し、積み立てNISAで最も人気の商品。1本で全世界に分散できる。
マクロ経済 MACRO ECONOMICS → 詳細は macro-glossary.html
政策金利せいさくきんり
Policy Rate / FFレート
金利
中央銀行が設定する基準金利。すべての市場金利の基準。利上げ→借入コスト増・株安。利下げ→借入コスト減・株高。米国のFFレートが世界の金融市場に最大の影響を持つ。
FRB / FOMC
Federal Reserve / Federal Open Market Committee
機関
米国の中央銀行（FRB）とその政策決定会合（FOMC）。年8回開催。金利決定・量的緩和（QE）・量的引き締め（QT）を決定する。議長会見の一言で世界の相場が動く。
CPI（消費者物価指数しょうひしゃぶっかしすう）
Consumer Price Index
指標
インフレの温度計。FRBが最重視する指標の一つ。予想を上回ると利上げ期待→株安・ドル高。下回ると利下げ期待→株高・ドル安。発表日は相場が大きく動くため要注意。
VIX（恐怖指数きょうふしすう）
CBOE Volatility Index
指標
S&P500オプション価格から算出される「市場の不安感」指標。VIX20以下=安定。VIX30超=警戒。VIX50超=パニック。株価と逆相関することが多い。デイトレでは毎朝確認する必須指標。
DXY（ドルインデックス）
US Dollar Index
為替
主要6通貨に対するドルの強さを指数化。DXY↑=ドル高→新興国・コモディティ・金に逆風。DXY↓=ドル安→これらに追い風。リスクオン・オフの代理指標としても使われる。
リスクオン・リスクオフ
Risk-On / Risk-Off
市場心理
リスクオン：投資家が積極的にリスク資産（株・新興国）を買う局面。リスクオフ：不確実性が高まり安全資産（金・円・米国債）に逃避する局面。地政学リスクや悪い指標がリスクオフの引き金になる。
逆イールドカーブ
Inverted Yield Curve
金利
短期金利＞長期金利になる異常な状態。景気後退の前兆シグナルとして有名。米2年債と10年債の逆転が市場で最も注目される。逆イールドが続いた後に景気後退が来るパターンが多い。
QE・QT
Quantitative Easing / Tightening
政策
QE（量的緩和）：中央銀行が資産を購入して市場に資金を供給→株高。QT（量的引き締め）：資産を縮小して資金を回収→株に逆風。FRBのQE/QT転換は相場の大きな転換点になる。
円安・円高
Yen Depreciation / Appreciation
為替
円安：輸出企業↑・インバウンド↑・輸入コスト↑・実質賃金↓。円高：輸出企業↓・輸入コスト↓・家計の購買力↑。デイトレでは朝の円相場チェックがセクター選択の第一歩。
ゴールド（金）
Gold / XAU
コモディティ
実質金利と逆相関する安全資産。実質金利↓→金↑。地政学リスク・インフレへのヘッジ。ドル安局面でも上昇しやすい。ポートフォリオの5〜15%程度組み入れることでリスク軽減効果がある。
