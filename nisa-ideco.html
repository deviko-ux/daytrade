02
TAX BENEFIT
税制メリットの違いを理解する
NISA の税制
売却益・配当が非課税

投資して得た利益（売却益・配当）に対して課税されない。掛け金に対する控除はないが、シンプルで分かりやすい。いつでも売れる流動性も高い。

iDeCo の税制（3段階の優遇）
3つすべてで優遇される

①掛け金が全額所得控除→ 毎年の所得税・住民税が減る。②運用益が非課税→ NISAと同じ。③受取り時に控除あり→ 退職所得控除・公的年金等控除が使える。

// iDeCoの所得控除メリット試算（年収600万円・月2.3万円の場合）
// ────────────────────────────────────────
年間掛け金：27.6万円
所得税率（20%）での節税：27.6万 × 20% = 約5.5万円/年
住民税率（10%）での節税：27.6万 × 10% = 約2.8万円/年
年間節税合計：約8.3万円
// 30年継続なら節税総額 → 約249万円（運用益とは別）
POINT — iDeCo の落とし穴
受取り時に課税されるが、退職所得控除（勤続年数 × 40万円、20年超は70万円）が使える。ただし、会社の退職金との控除の重複・会社員退職後2〜5年以内の引き出しなど、複雑な条件がある。受取り方は事前に確認が必要。
03
PRIORITY
どちらを優先すべきか
BASIC ANSWER
原則：まずiDeCoで「節税」、次にNISAで「資産形成」
ただし、収入が低い・近い将来に大きな支出がある・流動性が欲しい場合はNISA優先が合理的。
年収300万以下？
→ NISA優先。所得税が少なく、iDeCoの控除メリットが小さい。流動性が高いNISAで柔軟に。
近い将来に大きな出費？
→ NISA優先。iDeCoは60歳まで引き出せないため、住宅購入・教育費など近い将来の支出がある場合はNISAが安全。
年収400万以上で余裕がある？
→ iDeCoを月上限まで + 残りをNISAへ。所得控除の節税効果が大きく、iDeCoを最大活用するのが最も効率的。
自営業・フリーランス？
→ iDeCo最優先。会社員より掛け金上限が高く（月6.8万円）、国民年金のみで老後保障が薄い分、iDeCoでの積み上げが重要。
公務員？
iDeCoの上限が月1.2万円と低め。NISA中心で運用し、iDeCoは補完的に使う戦略が現実的。

06
WITHDRAWAL
iDeCo の受け取り方（出口戦略）
一時金受取
退職所得として受取

一括で受け取る方法。退職所得控除（20年超：800万円 + 70万円×超過年数）が適用される。控除内に収まれば税負担がほぼゼロ。会社の退職金との調整が必要。

年金受取
雑所得として分割受取

5〜20年の分割払い。公的年金等控除が適用される。ただし複数の年金受取がある場合、合算で課税されることに注意。社会保険料への影響も考慮が必要。

一時金 + 年金の併用
最も柔軟な戦略

一部を一時金・残りを年金で受け取るハイブリッド。それぞれの控除を組み合わせることで税負担を最小化できる可能性がある。退職時に税理士への相談が望ましい。

WARNING — 退職金との重複問題
iDeCoの一時金と会社の退職金は、同じ年に受け取ると退職所得控除が重複できないケースがある。iDeCoを先に受け取った場合、その後19年間（一部4年間）は会社退職金の控除が制限される。受け取りタイミングは事前に会社のHR・税理士に確認すること。
